# React Shop Design System

Welcome to the React Shop Design System documentation.

## Overview

This design system provides a comprehensive set of components for building modern e-commerce applications.

## Component Categories

### Atoms
Basic building blocks like buttons, inputs, and text components.

### Molecules
Composite components built from atoms, like cards and ratings.

### Organisms
Complex components like modals and product cards.

### Layout
Layout components like headers, footers, and navigation.

## Getting Started

Browse the components in the sidebar to see live examples, documentation, and interactive controls.

## Design Principles

- **Framework Agnostic**: Components are pure presentational elements
- **Type Safe**: Full TypeScript support
- **Accessible**: WCAG compliant components
- **Themeable**: Powered by Tailwind CSS
- **Modular**: Import only what you need
